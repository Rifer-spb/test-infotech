Установка
------------

Запустить контейнеры docker:

~~~
make docker-up
~~~

Перейти в cli:

~~~
make cli
~~~

Установить зависимости:

~~~
composer install
~~~

Выполнить миграции:

~~~
php yii migrate
~~~

Готово - <a href ="http://localhost:8080/" target="_blank">http://localhost:8080/</a>
